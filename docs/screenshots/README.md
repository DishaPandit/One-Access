# Screenshots

Add your app screenshots here to display in the main README.

## Naming Convention

- `access.png` - Access tab with QR code
- `time.png` - Time tracking screen
- `delegate.png` - Delegation screen
- `visitor.png` - Visitor pass screen
- `qr.png` - QR code close-up
- `session.png` - Active session view

## Specifications

- **Format:** PNG
- **Size:** Optimized for web (< 500KB per image)
- **Dimensions:** 1080x2400 (portrait) or crop to relevant area
- **Background:** Use actual app screenshots or mockups

## Tips

1. Take screenshots on a real device or emulator
2. Sign in before capturing
3. Show actual data (not empty states)
4. Crop to remove status bar if desired
5. Use high-resolution device for crisp images

## Tools

- **Android Studio:** Built-in screenshot tool (logcat toolbar)
- **ADB:** `adb exec-out screencap -p > screenshot.png`
- **Device:** Use phone's built-in screenshot feature
